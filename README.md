## Getting started
#### Clone repository
```
    git clone git@github.com:lkbrennan97/IMT-542-Transit-Lab.git
```
#### setup
```
    cd IMT-542-Transit-Lab
    python -m venv venv
    source venv/bin/activate
```

#### install requirements 
```
    pip install -r requirements.txt
```
#### unzip data file
```
    unzip data/kc_metro.zip
```
#### run jupyter notebook
If you want to run the Jupyter notebook from the command line
```
    pip install notebook
    
    jupyter notebook workflow.ipynb
```
